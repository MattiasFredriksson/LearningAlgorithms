###### Uses the scikit-learn package in python.
###### To install the package go to:
###### http://scikit-learn.org/stable/install.html
###### Make sure all pre-requisites are installed!


To run a test, call:
python assignment.py

To run test for a specific metric add ONE of the following types:
0: F-measure, 1: Accuracy, 2: Training time

Example (F-measure):
python assignment.py 0

##### Extra:
To visualize the result (compute the friedman test for the metric written in
result.txt), call:
python friedman.py


The data used for the report can be found in ./results/
(not possible to pass as parameters for friedman)
